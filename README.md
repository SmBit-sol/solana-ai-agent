## 🧠 What it does

- 📈 Monitors token prices on Solana devnet in real time
- 🔄 Executes swaps autonomously via Jupiter DEX
- 🎯 Places and manages prediction market positions
- 🤖 Powered by Claude AI for autonomous decision making
- 🔐 Non-custodial — you own your keys

## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/SmBit-sol/solana-ai-agent
cd solana-ai-agent

# Install Python deps
pip install -r requirements.txt

# Setup env
cp .env.example .env
# Add your ANTHROPIC_API_KEY to .env

# Run agent
python agent/main.py
```
